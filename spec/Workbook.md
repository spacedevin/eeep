# EPPlus Workbook Specification

## Core Structure
- Workbook Properties
  - Title
  - Subject
  - Author
  - Comments
  - Keywords
  - Category
  - Status
  - Version
  - Revision
  - Created/Modified dates
  - Application version

## Workbook Features
- Sheet Management
  - Add/remove sheets
  - Copy/move sheets
  - Hide/unhide sheets
  - Sheet order
  - Sheet protection
- Name Management
  - Named ranges
  - Named formulas
  - Workbook scope
  - Sheet scope
- Protection
  - Structure protection
  - Window protection
  - Password protection
  - Digital signatures

## Implementation Priority
1. Core properties
2. Sheet management
3. Name management
4. Protection features

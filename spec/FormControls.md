# EPPlus Form Controls Specification

## Control Types
- Basic Controls
  - Button
  - Checkbox
  - Combobox
  - Listbox
  - Textbox
  - Spinbutton
- Advanced Controls
  - Group box
  - Label
  - Radio button
  - Scrollbar
  - Image

## Control Features
- Properties
  - Name/Caption
  - Size/Position
  - Value/State
  - Enabled/Disabled
  - Visible/Hidden
- Events
  - Click
  - Change
  - Enter/Exit
  - KeyPress
  - MouseMove

## Implementation Priority
1. Basic controls
2. Control properties
3. Event handling
4. Advanced features

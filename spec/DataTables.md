# EPPlus Data Tables Specification

## Table Types
- Single Variable Tables
  - Row input
  - Column input
  - Formula results
  - Custom inputs
- Multiple Variable Tables
  - Two-variable tables
  - Cross tables
  - Sensitivity tables
  - Result matrices

## Table Features
- Input Management
  - Input cells
  - Input ranges
  - Input validation
  - Input formatting
- Result Management
  - Result cells
  - Result ranges
  - Result formatting
  - Result validation
- Calculation Control
  - Auto calculate
  - Manual calculate
  - Recalculation
  - Dependencies

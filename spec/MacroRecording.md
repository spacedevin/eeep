# EPPlus Macro Recording Specification

## Recording Types
- Action Recording
  - Cell changes
  - Format changes
  - Formula changes
  - Selection changes
- Operation Recording
  - Sheet operations
  - Range operations
  - Data operations
  - UI operations

## Recording Features
- Recording Control
  - Start/Stop
  - Pause/Resume
  - Cancel
  - Modify
- Code Generation
  - VBA code
  - TypeScript code
  - Function code
  - Event handlers

## Implementation Priority
1. Basic recording
2. Action tracking
3. Code generation
4. Advanced features

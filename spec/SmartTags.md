# EPPlus SmartTags Specification

## Tag Types
- Built-in Tags
  - Person names
  - Places
  - Dates
  - Financial data
- Custom Tags
  - User-defined
  - Application-specific
  - Domain-specific
  - Context-aware

## Tag Features
- Tag Properties
  - Name
  - Value
  - Type
  - Metadata
- Tag Operations
  - Add tag
  - Remove tag
  - Update tag
  - Query tags
